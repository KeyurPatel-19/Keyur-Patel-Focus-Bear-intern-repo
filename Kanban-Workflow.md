# Agile Workflows & Kanban

## Research & Learn

### How does a Kanban board work, and how does it help manage workflow?

A Kanban board is a visual management tool that organizes tasks using cards, which move across columns representing different stages of progress. This system enables me to clearly identify pending, ongoing, and completed tasks, thereby maintaining organization and facilitating efficient tracking.

### What do the different columns on a Kanban board represent? (e.g., Backlog, In Progress, Blocked, Done)

Each column shows a stage of work. For example:
- Not Started – Tasks I haven’t started yet
- In Progress – Tasks I am currently working on
- Blocked – Tasks that are stuck
- Ready for Review – Completed tasks waiting for feedback
- Done / Approved – Fully completed tasks
These columns help me understand the current status of my work.

### How do tasks move through the board, and who is responsible for updating them?

Tasks move from left to right as I complete each stage.
For example:
Not Started → In Progress → Ready for Review → Done
I am responsible for updating the task status when I make progress. This keeps the board accurate.

### What are the benefits of limiting work in progress (WIP)?

Limiting WIP means I don’t work on too many tasks at the same time. It helps me in Stay focused, Finish tasks faster, Avoid stress, Improve work quality.

## Reflection

### How does Kanban help manage priorities and avoid overload?

Kanban helps me clearly see which tasks are important. When everything is visible, I can focus on finishing one task before starting another. This prevents overload and improves productivity.

### How can you improve your workflow using Kanban principles?

I can improve my workflow by:
- Breaking big tasks into smaller ones
- Updating the board regularly
- Working on only 1–2 tasks at a time
- Reviewing my board daily
This helps me stay organized and consistent.
