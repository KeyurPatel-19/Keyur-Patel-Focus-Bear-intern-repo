# VS Code Productivity

## Navigation

- Cmd + P — Quick Open / Go to file
- Ctrl + G — Go to line
- Shift + Cmd + O — Go to symbol in file

## Editing

- Option + Click — Insert cursor
- Option + Cmd + Up / Down — Insert cursor above / below
- Cmd + D — Add selection to next find match
- Shift + Cmd + L — Select all occurrences of current selection
- Shift + Option + F — Format document

## Debugging

- F5 — Start / Continue debugging

## Terminal Commands

- Ctrl + ` — Show integrated terminal

## Which shortcuts do you think will be the most useful in your daily work?

Some of the shortcuts that will be most useful in daily work are  
Cmd + P, Cmd + D, Shift + Cmd + L, and multiple cursor shortcuts  
like Option + Click or Option + Cmd + Up/Down.

Cmd + P helps quickly open any file in a project without searching  
through folders. Cmd + D and Shift + Cmd + L are very useful when  
editing repeated variables, column names, or pieces of code.  
They allow multiple places in the code to be edited at the same time.

The multiple cursor feature is also very helpful when working with  
several lines of code, as it allows the same change to be applied  
across many lines quickly.

## How can keyboard shortcuts improve your productivity?

Keyboard shortcuts improve productivity by making common tasks  
much faster and smoother.

Instead of using the mouse to navigate menus or search for options,  
shortcuts allow actions like opening files, editing code, formatting  
documents, or opening the terminal to be done instantly.

This helps maintain focus while coding and reduces interruptions  
in the workflow. Over time, using shortcuts becomes a habit and  
can save a significant amount of time, especially when working on  
larger projects or handling repetitive coding tasks.
