# Set Up a Terminal Client & Get Comfortable with the Command Line

### Which terminal client did you choose? Why?

I chose iTerm2 with zsh on my Mac. I decided to use it because it looks cleaner and more modern than the default Terminal app. It also feels easier to use when working with development tools. Another reason I chose iTerm2 is that it supports many customizations, which can make the command line more comfortable to use for daily tasks.

### What customizations (if any) did you make?

I made a few simple customizations to keep my setup practical. I changed the appearance to a dark theme and added a few aliases in my `.zshrc` file, such as `gs` for `git status`, `ll` for `ls -la`, and `c` for `clear`. These small changes made the terminal quicker and easier to use.

### What was the most useful command you learned today?

The most useful command I learned today was ls -la. This command shows all files in a folder, including hidden files, along with detailed information. I found it very helpful because it makes it easier to understand the structure of a directory. I think it will be especially useful when working with Git repositories, configuration files, and coding projects.
